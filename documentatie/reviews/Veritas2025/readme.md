# Beveiligingsonderzoek door Veritas

Op verzoek van de Kiesraad heeft Veritas een beveiligingsonderzoek gedaan naar Abacus. Het doel van dit onderzoek was om Abacus alvast te toetsen voorafgaand aan de pentest voor de productieversie. Dit onderzoek is uitgevoerd onder leiding van de CISO, die een uitleg van de bevindingen en reactie heeft toegevoegd. Deze is naast het onderzoeksrappoert hier als PDF opgenomen.
