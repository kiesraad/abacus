# Rapport Adviescollege ICT Toetsing (december 2025)

Het Adviescollege ICT Toetsing heeft onderzoek gedaan naar Abacus. Het rapport is op 22 december 2025 gepubliceerd op de website van [AcICT](https://www.adviescollegeicttoetsing.nl/documenten/2025/12/08/abacus). Voor de volledigheid staat hier een lokale kopie. Er volgt een formele reactie van de Kiesraad, die is nu nog niet beschikbaar.
